# Estatidados Meetup

Estatidados é uma grande comunidade de estatística e ciência de dados no Brasil. Eles organizam um meetup online que reúne pesquisadores e desenvolvedores de destaque da comunidade de Estatística e Ciência de Dados para participar de um conjunto diversificado de palestras cobrindo as tendências atuais no campo de Data Science.

Wilson Rocha, o mantenedor do SysIdentPy, deu uma palestra no meetup sobre o SysIdentPy e o vídeo está disponível abaixo:

# Histórico de Alterações

!!! info "O changelog é mantido em inglês"

    O histórico de alterações (changelog) é mantido em inglês para facilitar a manutenção e consistência com as releases do GitHub. Para ver as alterações detalhadas, consulte a [versão em inglês do changelog](https://sysidentpy.org/changelog/changelog/) ou as [releases no GitHub](https://github.com/wilsonrljr/sysidentpy/releases).

## Resumo das Versões Recentes

### v0.6.0
- Introdução da classe `OFRBase` para algoritmos baseados em Error Reduction Ratio (ERR)
- Implementação do algoritmo Ultra Orthogonal Forward Regression (UOFR)
- Correção de typo: `Bersntein` → `Bernstein`
- Cobertura de testes aumentada para 92%
- Descontinuação do suporte ao Python 3.7

### v0.5.3
- Correção do bug da função base Bilinear para modelos com mais de 2 entradas

### v0.5.2
- Correção do bug das funções base Polynomial e Bilinear para modelos com mais de 3 entradas

### v0.5.1
- Correções menores e melhorias de estabilidade
